CSS Selectors

Selector Types

Selector Symbol Description
Type None A type selector matches the type of element in the HTML document.
Universal * Selects all elements of any type.
Class . Selects an HTML element by using the value of the class attribute.
ID # If an HTML element requires a unique style, the 'id' attribute is used.

Specificity

Selector Specifity Level Description
ID Most Specific ID's only style one element and will override styles of types and classes.
Class Semi Specific Classes are meant to be reused over many elements.
Type Least Specific Type selectors will apply style to all the elements with that tag name.
